# Os Churrasqueiros Beto e Mano

Aplicativo em React + Vite + Tailwind CSS para gerar orçamentos de eventos, salvar clientes e criar PDF profissional.

## Como rodar

```bash
npm install
npm run dev
```

Abra o endereço exibido no terminal, normalmente `http://localhost:5173`.

## Melhorias realizadas

- Logo aplicada na tela de abertura
- Logo aplicada no topo do aplicativo
- Logo aplicada no topo do PDF
- Tela de abertura com animação suave
- Paleta baseada na marca: preto, grafite, dourado e vermelho queimado
- Cards escuros com bordas arredondadas e sombras suaves
- Botões grandes, responsivos e com destaque visual
- Inputs modernos com foco suave
- Ícones com lucide-react
- PDF limpo com cabeçalho, dados do cliente, dados do evento, orçamento e valor final
- Removida a área de textos extras do PDF
- Mantida a lógica de cálculo existente
- Mantidas agenda, visualização, exclusão, histórico e criação de PDF
